Welcome To MyBlog

The MyBlog web application is developed as the example application presented in "The Well-Grounded Python Developer" book.

The MyBlog application evolves over the course of chapters in the book as new features and functionality are added.

Most of the features added over time are intended to present information, tools and techniques to help developers increase their ability to handle larger projects and think in big picture ways.

The ultimate goal of the book is to help you translate that picture into a solid, maintainable application implementation.

From this page you can navigate to the MyBlog content which is the primary functionality the application provides.
